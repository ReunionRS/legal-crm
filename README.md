# Legal CRM

`Legal CRM` — это приложение для ведения клиентской базы юриста или небольшой юридической практики.

Система помогает хранить клиентов в одном месте, отслеживать этап работы по каждому обращению и получать уведомления в Telegram.

## Что умеет приложение

- авторизация администратора
- добавление новых клиентов
- редактирование карточки клиента
- удаление клиентов
- смена статуса клиента
- поиск по клиентской базе
- просмотр общей статистики на панели
- экспорт клиентской базы в Excel
- Telegram-уведомления о новых клиентах и изменении статуса
- команда `/stats` в Telegram-боте
- установка как PWA на телефон или компьютер
- Android APK для мобильного использования

## Какие данные ведутся по клиенту

Для каждого клиента можно хранить:

- имя
- телефон
- статус дела

Статусы работы:

- `Новый`
- `В работе`
- `Закрыт`

## Как работать с приложением

### 1. Вход администратора

Администратор входит в систему через страницу авторизации.

После входа открывается панель управления.

### 2. Панель

На панели отображается:

- общее количество клиентов
- количество новых обращений
- количество клиентов в работе
- количество закрытых дел
- последняя активность по клиентам

### 3. Клиенты

Во вкладке `Клиенты` можно:

- просматривать список клиентов
- искать клиента по имени, телефону или статусу
- изменить статус клиента
- открыть карточку клиента
- отредактировать клиента
- удалить клиента
- экспортировать базу клиентов в Excel

### 4. Добавление клиента

При создании клиента заполняются:

- имя клиента
- телефон
- статус дела

Телефон вводится с маской формата:

`+7 (999) 999-99-99`

### 5. Настройки

Во вкладке `Настройки` настраивается Telegram-интеграция.

Можно:

- включить или выключить уведомления
- сгенерировать код привязки
- привязать Telegram-аккаунт к CRM
- отвязать Telegram-аккаунт

Для привязки используется бот:

`@cr222mbot`

### 6. Как подключить Telegram

1. Откройте настройки
2. Нажмите `Сгенерировать код`
3. Откройте бота `@cr222mbot`
4. Отправьте команду `/start`
5. Отправьте боту код из CRM

После этого бот будет присылать уведомления и принимать команду:

`/stats`

## Для чего подходит это приложение

`Legal CRM` подходит для:

- частного юриста
- помощника юриста
- небольшой юридической команды
- ведения базы обращений и контроля этапов работы

## Основная идея

Приложение сделано как простая CRM без лишних разделов: только клиенты, статусы, уведомления и быстрый доступ к базе в вебе, PWA и Android.
